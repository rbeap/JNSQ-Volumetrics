# JNSQ Volumetrics
Volumetrics support for JNSQ with new Scatterer configs

![image](https://github.com/rbeap/JNSQ-Volumetrics/assets/170141846/a4d0031c-9097-489a-9cfc-8947376d9094)


=====================
# DEPENDENCIES
=====================
- EVE Redux Release 4
- ModuleManager
- Scatterer
- JNSQ (obviously)

=====================
# INSTALLATION
=====================

- Drop JNSQVolumetrics into GameData.
